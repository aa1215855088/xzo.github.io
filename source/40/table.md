首先，两个循环，就能打出一个矩形：

代码:

```
public class Table {

    public static void main(String[] args) {
        
        System.out.println("==============================99乘法表开始==============================");
        for(int i = 1; i < 10; i++) {
            for(int j = 1; j < 10; j++) {
                String output = j + "X" + i + "=" + ( i * j) + "  ";
                System.out.print(output);
            }
            // 这里是为了换行
            System.out.println("");
        }
        System.out.println("==============================99乘法表结束==============================");
        
    }
}

```

但是矩形肯定是不对的，多打印出来了一一半，观察一下，***乘数不应该比被乘数大，那么就在乘数等于被乘数***的时候，就不再打印之后的了，代码里只要在第二个循环里面的开始前，加入判断***如果乘数小于等于被乘数，则打印***

代码：

```
public class Table {

    public static void main(String[] args) {
        
        System.out.println("==============================99乘法表开始==============================");
        for(int i = 1; i < 10; i++) {
            for(int j = 1; j < 10; j++) {
                // 这句判断
                if(j <= i) {
                    String output = j + "X" + i + "=" + ( i * j) + "  ";
                    System.out.print(output);
                }
            }
            // 这里是为了换行
            System.out.println("");
        }
        System.out.println("==============================99乘法表结束==============================");
        
    }
}
```

还有一种把for换成 ***递归*** 的方法，要看懂这个，你得熟悉 java 的 “方法“， “递归”的概念，东西先放在这里，以后你可以看看，先看看代码：

```
public class Table {

	public static void main(String[] args) {

		System.out.println("==============================99乘法表开始==============================");
		// 打印 i 
		printI(1);
		System.out.println("==============================99乘法表结束==============================");

	}

	private static void printI(int i) {
		if (i >= 10) {
			// 递归出口，意思是结束递归
			return;
		}
		
		// 打印 j
		printJ(i, 1);
		// 换行
		System.out.println("");
		
		i++;
		// 继续打印 i
		printI(i);
	}

	private static void printJ(int i, int j) {
		if(j >= 10) {
			// 递归出口，意思是结束递归
			return;
		}
		String output = j + "X" + i + "=" + (i * j) + "  ";
		System.out.print(output);
		if (j >= i) {
			// 乘数大于被乘数，则返回，不继续打印
			return;
		}
		
		j++;
		// 继续打印 j
		printJ(i, j);
	}

}

```

递归的概念：

> 程序调用自身的编程技巧称为递归（ recursion）
